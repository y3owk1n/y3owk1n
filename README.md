## A vim lover with a macOS

I built programs mainly for my own use cases. Currently manages the whole system via nix darwin & home manager on my macOS. Here's my [system repo](https://github.com/y3owk1n/nix-system-config-v2).

### What I've built and still using

#### Neovim plugins

- [undo glow](https://github.com/y3owk1n/undo-glow.nvim) - adds beautiful visual feedback to your edits in Neovim
- [time machine](https://github.com/y3owk1n/time-machine.nvim) - take control of your edit history with an interactive timeline, diff previews, taggings, live reloading trees and cleanup functions
- [dotmd](https://github.com/y3owk1n/dotmd.nvim) - keep all your notes, todos, and journals inside Neovim without ever leaving the editor
- [base16 pro max](https://github.com/y3owk1n/base16-pro-max.nvim) - base16 for modern Neovim
- [warp](https://github.com/y3owk1n/warp.nvim) - lightweight project-local file list for Neovim
- [tailwind autosort](https://github.com/y3owk1n/tailwind-autosort.nvim) - format tailwind classes without `prettier-plugin-tailwindcss`
- [notifier](https://github.com/y3owk1n/notifier.nvim) - modern, feature-rich notification system for Neovim

#### Neovim utils

- [nvs](https://github.com/y3owk1n/nvs) - lightweight Neovim Version & Config Manager CLI tool that supports github actions

#### Keyboard navigation

- [neru](https://github.com/y3owk1n/neru) - keyboard-driven navigation for macOS, open source alternative to homerow, vimac and shortcat
